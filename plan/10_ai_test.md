## 10.10 보안 점검

- [ ] JWT 토큰 없이 보호된 API 접근 시 401 반환
- [ ] 다른 사용자의 데이터 접근 불가 (권한 분리)
- [ ] 외부 계정 비밀번호 AES-256 암호화 저장 확인
- [ ] SQL Injection 방지 (JPA Parameterized Query)
- [ ] XSS 방지 (프론트엔드 입력값 sanitize)
- [ ] CORS 설정 올바른 origin만 허용
- [ ] HTTPS 통신 확인 (Cloudflare Tunnel)
