# Step 6: 프론트엔드 연동 및 최종 서비스화 (반응형 웹앱)

## 6.1 전체 페이지 구조

```
┌────────────────────────────────────────────────────┐
│              상단 네비게이션 바                        │
│  "이끼잡" 로고  │  탭: 사람인 / 잡플래닛  │  프로필 아이콘 │
└────────────────────────────────────────────────────┘
                         │
         ┌───────────────┼───────────────┐
         ▼               ▼               ▼
   ┌───────────┐  ┌───────────┐  ┌───────────────┐
   │  메인 페이지  │  │ 프로필 관리  │  │ 프로젝트/템플릿  │
   │ (공고 목록)  │  │   페이지    │  │   관리 페이지    │
   └──────┬────┘  └───────────┘  └───────────────┘
          │
          ▼
   ┌─────────────────┐
   │  공고 상세 페이지   │
   │  (기업 분석 포함)   │
   └──────┬──────────┘
          │
          ▼
   ┌─────────────────────┐
   │  Preview/Edit 페이지  │
   │  (자소서/포폴 검토)   │
   │  (2분할 레이아웃)     │
   └─────────────────────┘
```

---

## 6.2 상단 네비게이션 바

| 요소 | 위치 | 동작 |
|------|------|------|
| **"이끼잡" 로고/텍스트** | 좌측 | 클릭 시 홈(메인 페이지)으로 이동 |
| **사이트 탭** | 중앙 | 사람인 / 잡플래닛 탭 전환 |
| **검색창** | 중앙~우측 | 키워드 검색 (현재 탭 공고만 결과) |
| **알림 아이콘** | 우측 | 미확인 알림 배지 표시 |
| **프로필 아이콘** | 우측 | 드롭다운: 프로필, 프로젝트, 템플릿, 설정, 로그아웃 |

---

## 6.3 메인 페이지 (공고 목록)

### 레이아웃 모드
| 모드 | 데스크탑 | 모바일 | 표시 정보 |
|------|---------|--------|----------|
| **카드형** (기본) | 3열 그리드 | 1열 스택 | 회사 로고, 제목, 회사명, 기술 스택 태그, 적합률, 마감일, 지원 상태 |
| **리스트(테이블)형** | 풀 너비 테이블 | 1열 간소화 | 제목, 회사명, 지역, 경력, 적합률, 마감일, 지원 상태 |
| **토글 버튼** | 우상단 | 우상단 | 카드 ↔ 리스트 전환 |

### 사이트별 탭 분리
- **사람인 탭**: 사람인 전용 카드/테이블 UI 템플릿으로 렌더링
- **잡플래닛 탭**: 잡플래닛 전용 카드/테이블 UI 템플릿 (기업 평점 등 잡플래닛 특화 정보 포함)
- **데이터 구조가 다르므로 탭별 전용 컴포넌트** 사용
- 검색은 전체 탭 공통이되, **결과는 현재 선택된 탭의 공고만** 표시

### 지원 상태 표시
| 상태 | 라벨 색상 | 조건 |
|------|----------|------|
| **미지원** | 회색 | 기본 |
| **대기 중 (AI 생성 완료)** | 파란색 | AI 자소서/포폴 사전 생성 완료, 검토 대기 |
| **지원 완료** | 초록색 | 자동 지원 성공 (즉시지원) |
| **수동 지원 완료** | 초록색 (아이콘 다름) | 사용자가 "지원 완료" 버튼으로 수동 표시 (홈페이지 지원) |
| **지원 실패** | 빨간색 | 자동 지원 실패 + 재시도 버튼 |
| **마감** | 어두운 회색 | 공고 마감 |

> **홈페이지 지원 공고**는 자동 확인 불가 → **사용자가 직접 "지원 완료" 버튼**으로 수동 표시

## 6.4 고급 필터링/정렬

사람인/잡플래닛 사이트의 실제 필터 항목을 **벤치마킹**하여 동일하게 구현:

### 필터 항목
| 필터 | 설명 | UI 형태 |
|------|------|---------|
| **직종** | IT/개발, 디자인, 마케팅 등 | 드롭다운 (멀티 선택) |
| **개발 분야** | 백엔드, 프론트엔드, 풀스택, DevOps 등 | 체크박스 그룹 |
| **학력** | 학력 무관, 초대졸, 대졸 등 | 드롭다운 |
| **경력** | 신입, 1~3년, 3~5년 등 | 범위 슬라이더 |
| **연봉** | 전체, 2500~3000, 3000~4000 등 | 범위 슬라이더 |
| **근무 지역** | 시/도 → 구/군 계층 선택 | 2단계 드롭다운 |
| **지원 방식** | 즉시지원, 홈페이지지원, 이메일 | 체크박스 |
| **적합률** | 80% 이상만 보기 등 | 범위 슬라이더 |
| **지원 상태** | 미지원, 대기 중, 지원 완료 등 | 체크박스 |

### 정렬 옵션
| 정렬 기준 | 기본값 | 순서 |
|----------|--------|------|
| **최신순** | ✅ | 내림차순 |
| **적합률순** | | 내림차순 |
| **마감임박순** | | 오름차순 |
| **연봉순** | | 내림차순 |

## 6.5 반응형 웹앱 (PWA 고려)

### 브레이크포인트
| 크기 | 조건 | 레이아웃 |
|------|------|---------|
| **모바일** | ~768px | 1열 카드, 햄버거 메뉴, 하단 탭바 |
| **태블릿** | 769~1024px | 2열 카드, 사이드 필터 접기 |
| **데스크탑** | 1025px~ | 3열 카드, 사이드 필터 상시 표시 |

### PWA 기능 (고려 사항)
- 오프라인 기본 캐싱 (Service Worker)
- 홈 화면 추가 아이콘
- 푸시 알림 (디스코드 외 추가 알림 채널)
